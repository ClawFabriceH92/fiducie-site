# Fiducie Site Prototype

Prototype statique orienté acquisition pour un site sur la fiducie en France, destiné à un cabinet d’expertise comptable / CAC.

## Positionnement
Pilotage fiducie pour dirigeants, DAF et opérations sensibles.

## Pages incluses
- Landing page / accueil longue
- Sections : compréhension, fiducie-sûreté, cas d’usage, différenciation, lead magnet, contact

## Déploiement
Ce dépôt est prévu pour un déploiement simple sur GitHub Pages.
